# Speak Yodish
`Speak Yodish, you will!` <br/><br/>
A fun translation site that converts English text into the language of Yoda i.e., Yodish!

## How is it built?
- HTML
- CSS
- Vanilla JavaScript
- [funtranslations API](https://funtranslations.com/)
